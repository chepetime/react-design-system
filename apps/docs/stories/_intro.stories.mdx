# chepe Design Language System

Quick explanation of the DLS

<Button>Hi</Button>

```tsx
import React from "react";
import { Button } from "@chepe/ui";

function App() {
  return (
    <div>
      <Button>Hi</Button>
    </div>
  );
}

export default App;
```

## Install

Install React Components

```sh
npm install @chepe/ui
```

Install Developer-ready Design Tokens

```sh
npm install @chepe/@chepe/tokens
```

Install React Icons (WIP)

```sh
npm install @chepe/chepe-icons
```
