# chepe Design Tokens

> UI Design Tokens in css, js, ts, json, sass and less

## Getting Started

Copy the .env.example and add your API Key;

```sh
cp .env.example .env
code .env
```

Update the tokens

```sh
npm run token-engine:fetch
```
